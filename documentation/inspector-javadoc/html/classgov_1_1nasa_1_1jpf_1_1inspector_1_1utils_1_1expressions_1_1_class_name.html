boolean gov.nasa.jpf.inspector.utils.expressions.ClassName.isSameClass ( ClassInfo  ci)

Returns true if this expression represents the specified class.

Comparison is done based on string names, except that the class name expression might not be a simple name but can be more complex (see the documentation on expressions; this is taken into account here).

Performs caching.

Parameters
